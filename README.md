# GHHS-EC&DC - Secure Encryption/Decryption Tool

## ОПИСАНИЕ
GHHS-EC&DC - это программа для безопасного шифрования и дешифрования текстовых данных и файлов с графическим пользовательским интерфейсом. Приложение обеспечивает военный уровень защиты данных с использованием алгоритма AES-256-GCM.

**Основные характеристики:**
- Алгоритм шифрования: AES-256-GCM (Galois/Counter Mode)
- Производная ключа: PBKDF2-HMAC-SHA256 с 100000 итераций
- Аутентифицированное шифрование с проверкой целостности данных
- Поддержка текстовых данных и файлов любого формата

**Технические параметры:**
- Размер ключа: 256 бит
- Размер блока: 128 бит  
- Соль: 16 байт
- Одноразовый номер (nonce): 12 байт
- Аутентификационный тег: 16 байт

## ИНСТРУКЦИЯ ПО УСТАНОВКЕ И ИСПОЛЬЗОВАНИЮ

### Установка
1. Убедитесь, что установлен Python версии 3.8 или выше
2. Скачайте файлы программы в отдельную директорию
3. Установите зависимости: `pip install -r requirements.txt`
4. Запустите программу: `python main.py`

### Использование

**Шифрование текста:**
1. Выберите вкладку "ШИФРОВАНИЕ"
2. Введите ключ шифрования в соответствующее поле
3. Введите текст для шифрования в поле ввода
4. Нажмите кнопку "Зашифровать"
5. Скопируйте результат из поля вывода

**Дешифрование текста:**
1. Выберите вкладку "ДЕШИФРОВАНИЕ"
2. Введите корректный ключ шифрования
3. Введите зашифрованные данные в поле ввода
4. Нажмите кнопку "Расшифровать"
5. Получите исходный текст в поле вывода

**Работа с файлами:**
1. Используйте кнопки "Выбрать файл для шифрования" или "Выбрать зашифрованный файл"
2. Выберите соответствующий файл через диалоговое окно
3. Введите ключ шифрования
4. Нажмите соответствующую кнопку для шифрования/дешифрования
5. Сохраните результат операции

**Элементы управления интерфейсом:**
- "Сменить Тему" - переключение между светлой и темной темой оформления
- "Сменить Язык" - переключение между русским и английским интерфейсом
- "Очистить всё" - сброс всех полей текущей вкладки

## ОСОБЕННОСТИ

### Безопасность
- Военный уровень шифрования AES-256-GCM
- Аутентифицированное шифрование с проверкой целостности данных
- Стойкий алгоритм наследования ключей PBKDF2
- Уникальные параметры для каждой операции шифрования
- Секретное удаление ключей из памяти после использования

### Пользовательский интерфейс
- Современный графический интерфейс на PyQt6
- Поддержка двух языков: русский и английский
- Две темы оформления: светлая и темная
- Раздельный интерфейс для шифрования и дешифрования
- Индикатор выполнения операций

### Функциональность
- Шифрование и дешифрование текстовых данных
- Шифрование и дешифрование файлов любого формата
- Проверка корректности ключа при дешифровании
- Обработка ошибок с информативными сообщениями

### Системные требования
- Операционные системы: Windows 10+, Linux, macOS 10.14+
- Python 3.8 или выше
- Минимум 512 МБ оперативной памяти
- 10 МБ свободного места на диске

### Зависимости
- PyQt6 == 6.6.1
- cryptography == 41.0.7

### Примечания по безопасности
- Программа не передает данные по сети, все операции выполняются локально
- Рекомендуется использовать ключи шифрования длиной не менее 16 символов
- Обязательно сохраняйте ключи шифрования для возможности восстановления данных
- Перед работой с важными данными рекомендуется провести тестовое шифрование
